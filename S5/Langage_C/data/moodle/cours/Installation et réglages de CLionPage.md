# Installation et réglages de CLionPage

Source: https://moodleng.insa-rennes.fr/mod/page/view.php?id=4593

##### Obtenir la licence gratuite : 2 possibilités  


##### 

  * En TP, utiliser la licence collective INSA: <http://liceduc-vmp-app02.educ.insa>


  * Créer un compte avec votre adresse mail INSA :  

    * Demander une licence académique gratuite : <https://www.jetbrains.com/fr-fr/community/education/#students>
    * La licence est disponible dans "Licenses" une fois que vous êtes connecté sur <https://www.jetbrains.com>
    * A la première utilisation de CLion, utiliser l'identifiant et le mot de passe de votre compte jetBrains pour bénéficier de votre licence étudiant



##### Sous Windows (comme en TP):

  * Si MinGW n'est pas installé, l'installer en suivant les instructions du paragraphe correspondant  

  * Aller sur le site <https://www.jetbrains.com/fr-fr/clion/>
  * Aller sur "Download", télécharger l'exe et le lancer :  



  1. Activer les options d'installation : laucher, update context menu, add launchers dir to the PATH
  2. Créer les associations pour les fichiers .c et .h files au moins (voire les autres)


  * Au premier lancement de CLion:  



  1. Choisir l'apparence et laisser les autres fonctionnalités par défaut
  2. Pour la licence, utiliser l'identifiant et le mot de passe de votre compte jetBrains pour bénéficier de votre licence étudiant
  3. Configure Toolchains : choisir MinGW. CLion devrait le détecter correctement. Si MinGW n'est pas installé sur votre ordi à ce moment-là, suivre les instructions du paragraphe correspondant. Une fois MinGW installé, vous pouvez reparaméter CLion en suivant les instructions suivantes: <https://www.jetbrains.com/help/clion/quick-tutorial-on-configuring-clion-on-windows.html#MinGW>  




##### Installation MinGW (nécessaire pour Windows)  


  * Aller sur <https://www.mingw-w64.org/downloads/#mingw-builds>, télécharger MingW-W64-builds et lancer l'exécutable. Lors de l'installation, laisser la configuration par défaut




##### Sous Linux/Mac

  * Aller sur le site <https://www.jetbrains.com/fr-fr/clion/>
  * Aller sur "Download" et suivre les instructions. Si un choix de compilateur est demandé, prendre gcc
  * Pour la licence, utiliser l'identifiant et le mot de passe de votre compte jetBrains pour bénéficier de votre licence étudiant
