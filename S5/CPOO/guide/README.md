# CPOO -- Conception et Programmation Orientees Objet

## Presentation du cours

**CPOO1** (Conception et Programmation Orientees Objet) est un cours de 3e annee en Informatique a l'INSA Rennes, enseigne par Arnaud Blouin. Le cours couvre les fondements de la programmation orientee objet en Java, la modelisation UML, les tests unitaires et les principes de conception logicielle.

**Format de l'examen** : examen ecrit de 2 heures (sur papier), documents de cours autorises. L'examen comporte systematiquement trois piliers : tests unitaires / analyse de code, diagrammes UML, et questions de theorie POO.
