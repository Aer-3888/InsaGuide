# ITI - Introduction aux Techniques de l'Ingenieur

This directory contains completed lab exercises (TPs) for the ITI course at INSA Rennes (3rd year CS). The course covers systems programming fundamentals: Linux, shell scripting, build tools, version control, and Python programming.

## Course Structure

### FUS Series (Fonctions et Utilisation des Systemes)
System-level programming and Unix tools

| TP | Directory | Topic |
|----|-----------|-------|
| TP1 | [tp1_shell_basics](tp1_shell_basics/) | Shell Basics & Text Processing |
| TP2 | [tp2_shell_advanced](tp2_shell_advanced/) | Advanced Shell Scripting |
| TP3 | [tp3_regex](tp3_regex/) | Regular Expressions & Patterns |
| TP4 | [tp4_gedcom](tp4_gedcom/) | Genealogy Data Parsing (Perl/Ruby) |
| TP5 | [tp5_build_tools](tp5_build_tools/) | Build Tools (make, gdb, gprof) |
| TP6 | [tp6_debugging](tp6_debugging/) | Debugging & Profiling |
| TP7 | [tp7_git](tp7_git/) | Git Version Control |

### LDS Series (Langages De Script)
Python programming and application development

| TP | Directory | Topic |
|----|-----------|-------|
| TP8 | [tp8_python_basics](tp8_python_basics/) | Python Basics & Sorting Algorithms |
| TP9 | [tp9_qt_gui](tp9_qt_gui/) | Qt GUI Programming (PyQt) |
| TP10 | [tp10_sqlite](tp10_sqlite/) | SQLite Database Management |
| TP11 | [tp11_web_scraping](tp11_web_scraping/) | Web Scraping (BeautifulSoup) |
| TP12 | [tp12_automation](tp12_automation/) | Process Automation & Scripting |

## Repository Organization

Each TP directory contains:
- `README.md` - Concepts walkthrough and learning objectives
- `src/` - Clean, commented solution code
- Original exercise PDFs (where applicable)

## Key Concepts Covered

### Systems Programming
- Unix shell (bash, tcsh)
- Process management
- File I/O and permissions
- Build automation (Makefiles)
- Debugging (gdb)
- Performance profiling (gprof)

### Version Control
- Git fundamentals
- Branching and merging
- Conflict resolution
- Collaborative workflows

### Python Development
- Object-oriented programming
- GUI development with Qt
- Database integration (SQLite)
- Web scraping and parsing
- Process automation

## Course Context

**Course**: ITI (formerly FUS/LDS)  
**Institution**: INSA Rennes  
**Level**: 3rd year Computer Science  
**Year**: 2020-2021

The ITI course provides foundational knowledge in systems programming that underpins more advanced topics in operating systems, networks, and software engineering.
