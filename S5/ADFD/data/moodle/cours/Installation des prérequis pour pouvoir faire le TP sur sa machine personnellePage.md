# Installation des prérequis pour pouvoir faire le TP sur sa machine personnellePage

Source: https://moodleng.insa-rennes.fr/mod/page/view.php?id=13926

Sous **Windows** :  


  * Téléchargez python3: <https://www.python.org/downloads/windows/> et installez-le. Lors de l'installation, choisissez d'installer l'utilitaire pip (gestionnaire de modules python) et d'ajouter python à votre PATH.
  * Ouvrez le dossier dans lequel vous allez faire votre TP
  * Faites un shift+clickdroit dans le dossier et choisissez "Ouvrir une fenêtre de commande ici"
  * Dans l'invite de commande éxécutez  _pip3 install pandas numpy sklearn nltk mlxtend matplotlib folium jupyter_  




##### **Sous Linux :**

  * Installez Python3: la plupart des distributions linux ont un paquet  _python3_. (Exécutez  _python3 --version_ dans un terminal pour vérifier si c'est installé)
  * Vérifiez que l'utilitaire  _pip3_ est installé en exécutant  _pip3 --version_
  * Si ce n'est pas le cas, votre distribution linux a probablement un paquet  _python3-pip_ ou similaire à installer
  * Installez les modules nécessaires pour le TP: dans un terminal exécutez en tant que root  _pip3 install pandas numpy sklearn nltk mlxtend matplotlib folium jupyter_

#### **Lancement du TP**

  * Télécharger les fichiers ci-joints et les mettre tous dans un même répertoire
  * Renommer le fichier  _TP_etud.pynb_ en  _TP_nom_prenom.ipynb_
  * Ouvrir une console dans le répertoire créé précédemment
    * Sous Windows: faire un shift+clickdroit dans le dossier et choisir "Ouvrir une fenêtre de commande ici"
  * Dans la console taper  _jupyter notebook_ : un onglet s'ouvre automatique dans votre navigateur
  * Dans l'onglet, cliquer sur  _pandas_intro.ipynb_ afin d'ouvrir le notebook d'exemple concernant la bibliothèque panda. Ce notebook vous aidera à comprendre comment fonctionne cette bibliothèque. Ce document n'est pas à compléter ou à rendre.
  * Dans l'onglet cliquer sur  _TP_nom_prenom.ipynb_ afin d'ouvrir le notebook concernant le sujet du TP.  


  


  * Dans le notebook vous retrouvez 2 types de cases
    * Les "markdown" qui sont du texte.
    * Les "code" qui sont du code que vous pouvez exécuter.  




NOTE : Avant la première exécution faire "kernel -> restart and clear output"
