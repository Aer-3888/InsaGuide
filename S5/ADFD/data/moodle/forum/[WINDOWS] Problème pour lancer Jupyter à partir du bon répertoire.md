Solution proposée par Guillaume :  
\- Dans le menu de windows, cliquer droit sur Jupyter  
\- Sélectionner "Ouvrir l'emplacement du fichier"  
=> cela vous ouvre une fenêtre avec le lien de Jupyter notebook  
\- Cliquer droit sur ce lien  
\- Sélectionner Propriétés  
\- Changer dans la cible à la fin => remplacer %USERPROFILE% par le chemin de votre dossier où vous avez le TP
