Le problème ne vient pas du notebook mais du Jupyter utilisé dans google colab.

Si les messages deviennent gênants et vous ralentissent vous pouvez les masquer (attention cela ne résout pas le problème, cela le masque) en ajoutant une cellule de code :

import warnings

warnings.filterwarnings(

"ignore",

category=DeprecationWarning,

module=r"jupyter_client\\..*"

)
