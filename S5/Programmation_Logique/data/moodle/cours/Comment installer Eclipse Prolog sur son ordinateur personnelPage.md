# Comment installer Eclipse Prolog sur son ordinateur personnelPage

Source: https://moodleng.insa-rennes.fr/mod/page/view.php?id=36662

Si vous voulez utiliser votre propre ordinateur :

  


  * installer votre version (Linux, macOS, Windows) depuis <https://eclipseclp.org/Distribution/CurrentRelease/>
  * pour exécuter des requêtes Prolog, utiliser au choix :  

    * la version en mode texte (**DosEclipse** sous Windows)  

    * la version en fenêtre graphique d'interaction (**TkEclipse** sous Windows)
  * pour éditer votre programme source, utiliser votre éditeur de textes favori puis :  

    * en mode texte, taper la requête de chargement dans la fenêtre texte : par exemple **["/Users/Etudiant/monTP.pro"]** sous Windows.  

    * en mode fenêtre graphique, cliquer dans le menu de la fenêtre graphique : **File > Compile...**
