# Programmation Logique - Travaux Pratiques

INSA Rennes, 3e annee Informatique - Annee 2020/2021

## Environnement

Tous les TP utilisent **ECLiPSe Prolog** (fichiers `.pl` et `.ecl`).

```bash
# Lancer ECLiPSe
$ eclipsep

# Charger un fichier
[eclipse 1]: ["fichier"].

# Executer les tests
[eclipse 2]: tests.
```

## Structure

```
tp/
  sujet.pdf              -- Cahier de TP (enonce complet)
  tp1/                   -- Interrogation style base de donnees
  tp2/                   -- Manipulation de termes construits (Poker)
  tp3/                   -- Listes
  tp4/                   -- Arbres binaires
  tp5/                   -- Arithmetique (Peano + binaire)
  tp6/                   -- Bases de donnees deductives
  tp8/                   -- Puzzle des dominos
```

## Resume des TPs

| TP | Sujet | Concepts cles |
|----|-------|---------------|
| 1 | Base de donnees (Valois + Menu) | Faits, regles, requetes, recursivite (ancetre) |
| 2 | Jeu de cartes (Poker) | Termes construits, unification, pattern matching |
| 3 | Listes | Recursion sur les listes, modes, ensembles |
| 4 | Arbres binaires | Structures recursives, parcours, ABR, cut |
| 5 | Arithmetique | Peano, representation binaire, contraintes |
| 6 | BDD automobile | Operations relationnelles, requetes recursives |
| 8 | Puzzle dominos | Generer et tester, backtracking, accumulateurs |
