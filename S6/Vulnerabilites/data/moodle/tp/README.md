## Setup

All TPs use standard Linux shell tools:
```bash
# Core utilities
ls, wc, cat, grep, sed, awk, tr, sort, uniq

# File inspection
file, stat, head, tail

# Text processing
cut, paste, join

# Compression
zip, unzip, tar, gzip
```

No special installation required - all tools available in standard Unix environments.
